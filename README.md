## 🚀 What this app does

gbl_root_canoe helps rooted Android users keep fingerprint payment working in a simple way. It is made for users who want root and still want to use fingerprint pay in daily life.

This app is for people who want:
- Root access on their device
- Fingerprint payment support
- A simple setup on Windows before moving files to the phone
- A clear path with no complex steps

## 🖥️ What you need

Before you start, make sure you have:

- A Windows computer
- A USB cable
- An Android phone with root access
- Fingerprint payment already set up in your payment app
- Enough free space to save the downloaded file

This app works best on modern Windows systems such as Windows 10 and Windows 11.
